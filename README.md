# 📊 YouTube Engagement Analysis Dashboard

## 📝 Project Overview
This project analyzes **viewer engagement across different YouTube categories** to understand how audiences interact with content.  
It demonstrates **data analysis, visualization, and insight generation skills**, making it suitable for LinkedIn or portfolio projects.

 Objective: Identify patterns in audience behavior and engagement across different categories of YouTube content.

---

## 🛠 Tools Used
- **Python (Jupyter Notebook)** – data cleaning, metric calculations
- **Tableau** – building visual dashboards and interactive charts

---

## 📂 Dataset
- **Source:** YouTube trending videos dataset (CSV)  
- **Key columns:**
  - `category_id` → YouTube video category
  - `views` → number of views
  - `likes` → number of likes
  - `dislikes` → number of dislikes
- **Calculated metrics:**
  - **Like–Dislike Ratio** = likes ÷ dislikes  
    *Shows positivity of audience reactions.*
  - **Likes per View** = likes ÷ views  
    *Shows how actively viewers like content.*
  - **Dislikes per View** = dislikes ÷ views  
    *Measures criticism levels.*
  - **Engagement Rate** = (likes + dislikes) ÷ views  
    *Overall interaction from viewers.*

---

## 🔍 Analysis Performed
- Compared **engagement metrics** across YouTube categories.  
- Applied **Top 10 ranking** for each metric to highlight leading categories.  
- Created a **scatter plot: Views vs Engagement Rate** to identify categories with high views but low engagement.  
- Built a **dashboard** combining all visualizations for clear interpretation.

## 💡 Key Insights
- **Mass categories** (Entertainment, Music) generate very high views but **lower engagement**.  
- **Niche categories** (Pets & Animals, Education) have **higher engagement rates**, meaning more active and loyal viewers.  
- **Comedy** attracts strong interaction but also more criticism.  
- **Scatter plot** reveals which categories have high views but low engagement — highlighting potential gaps in audience connection.

**✅ Takeaway:**  
Mass categories bring visibility, while niche categories build **stronger audience loyalty and interaction**.
